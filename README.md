# gomigemo

MigemoをGo言語で実装したものです。
辞書ファイルの構造を効率化し、メモリ

## 使い方

```
> go run main.go
> migemoKensaku
(migemo|みげも|ミゲモ|ｍｉｇｅｍｏ|ﾐｹﾞﾓ)(Kensaku|けんさく|ケンサク|建策|憲[作冊]|検索|献策|研削|羂索|Ｋｅｎｓａｋｕ|ｹﾝｻｸ)
```
